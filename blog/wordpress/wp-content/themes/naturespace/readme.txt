== Description ==

NatureSpace is an easily customizable WordPress multipurpose theme. It is a minimalistic, fully responsive and Retina ready theme that allows for easy viewing on any device.

== Installation ==

You can install this theme through the WordPress installer under "Appearance > Themes - Add New" by searching for it.
Alternatively, you can download the file, unzip it and move the extracted content via FTP to the "wp-content/themes" folder of your WordPress installation. Then, you will be able to activate the theme.

== Documentation ==

The complete documentation of NatureSpace names "documentation.html" and is located in the "docs" folder. It is also accessable via Theme Options Panel after installing and activating this theme.
